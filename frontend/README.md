# Gammaprep Frontend

Next.js frontend for the Gammaprep bootcamp website.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open http://localhost:3000

## Build for Production

```bash
npm run build
npm start
```

## Configuration

Update API endpoints in:
- `context/AuthContext.js`
- `pages/index.js`
- `pages/admin.js`

For production deployment, replace `http://localhost:5000` with your production API URL.
